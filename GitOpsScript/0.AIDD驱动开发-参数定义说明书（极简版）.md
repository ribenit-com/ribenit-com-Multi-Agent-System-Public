# 参数设置说明书（极简版）

```
举例：

prompts=(
"9f8b7c6a|DB_HOST|127.0.0.1"
"2a4d1e3f|DB_PORT|5432"
"5c7f8a1b|DB_USER|admin"
"6d3e2b9c|DB_PASS|password"
"1a9b3c7d|LOG_DIR|./logs"
"8e2f5b4a|ENV|dev"
)
```

| 随机标志 | 变量名 | 默认值 | 说明 |
|----------|--------|--------|------|
| `9f8b7c6a` | `DB_HOST` | `127.0.0.1` | 数据库主机地址 |
| `2a4d1e3f` | `DB_PORT` | `5432` | 数据库端口 |
| `5c7f8a1b` | `DB_USER` | `admin` | 数据库用户名 |
| `6d3e2b9c` | `DB_PASS` | `password` | 数据库密码 |
| `1a9b3c7d` | `LOG_DIR` | `./logs` | 日志存放目录 |
| `8e2f5b4a` | `ENV` | `dev` | 当前环境（dev/prod） |

---

> 💡 **说明**

| 字段 | 用途 |
|------|------|
| 随机标志 | AI 内部识别变量用 |
| 变量名 | 人类/脚本直接调用 |
| 默认值 | 初始化时赋值 |
